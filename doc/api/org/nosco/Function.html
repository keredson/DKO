Function.Custom
public abstract class Function<T>
extends java.lang.Object

A collection of SQL functions.   Use these to call functions on fields or values in the generated SQL.   For example: 

   import static org.nosco.Function.*;
   SomeClass.ALL.where(SomeClass.BIRTH_DATE.gt(NOW())).count()

would generate: 

   select * from some_class where birth_date > NOW()
