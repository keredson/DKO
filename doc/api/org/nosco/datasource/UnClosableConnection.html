Class UnClosableConnection

java.lang.Object
  org.nosco.datasource.UnClosableConnection
All Implemented Interfaces:
java.sql.Connection, java.sql.Wrapper
public class UnClosableConnection
extends java.lang.Object
implements java.sql.Connection

This class wraps a connection, but intercepts calls to close().   User is responsible for eventually closing the underlying connection either by keeping a reference to it or by calling closeUnderlying().
